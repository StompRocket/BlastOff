The Blast Off navbar is fully responsive, even on mobile. The default menu will disappear on small screans and be replaced by a menu icon which can be pressed to open the navigation pannel.

The navigation pannel functions similarly to a modal in that it is opened when active. Make sure to link whatever you want to open the nav to it's ID, otherwise it will not open.

The navigation panel is highly customizable, and can be altered to your preferences. For example, you could put a dropdown, sign up form, or anything else in the nav menu, to make it easier for mobile users to access.

Try resizing this screen, or viewing it on a mobile device to see the responsive navbar in effect.

Blast Off contains multiple button styles. Simply apply the button class to an element to use.

Blast Off features a fully responsive grid system. You can put as many columns as you want in a row and they will automatically fill the space. Each column will automatically turn into a row on small screens. Try resizing this window to see the result. The default break point is 700px. You can resize each column by using the following classes:

  • one-fifth
  • two-fifth
  • three-fifth
  • four-fifth
  • one-fourth
  • three-fourths
  • one-third
  • two-thirds


Boxes:

Blast Off contains multiple box styles. Each of which can be easily used on a div element. Add the container class to a box to give it padding, or use a div with the container class inside it to give padding to only certain elements.

Footer:

The Blast Off footer styling is minimal. It will, however stick to the bottom of the screen if the content is not as long as the window, and will not overlap the content. If you wish, you may add the material class to it to give it a slight top shadow, or material inner for a slight inner shadow, that gives the effect that the footer is slightly below the content. 

<footer class="footer">
  Material Design footer with inner shadow
</footer>
