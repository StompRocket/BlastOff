Buttons:





Blast Off contains multiple button styles. Simply apply the button class to an element to use. 
<button class="button" href="#!">Default Button</button>
<button class="button primary" href="#!"> Primary Button</button>
<button class="button dark" href="#!">Dark Button</button>
